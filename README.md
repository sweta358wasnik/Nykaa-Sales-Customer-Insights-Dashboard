
#  Nykaa Sales & Profitability Analysis Dashboard

##  Project Overview

This project presents an interactive Power BI dashboard built to analyze Nykaa’s sales performance, pricing strategy, margin trends, product ratings, and brand distribution.

The objective of this project is to transform raw e-commerce data into meaningful insights and business KPIs to support data-driven decision-making.

##  Business Objective

- Analyze total sales and profitability
- Identify high-margin products
- Compare brand and category performance
- Study rating and review trends
- Evaluate discount impact on margins

---

##  Tools & Technologies Used

- Microsoft Excel (Data Cleaning & Preparation)
- Power BI (Data Modeling & Dashboard Development)
- DAX (KPI & Margin Calculations)
